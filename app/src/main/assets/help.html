Weather App

The Application

This application allow user to get the forecast for a precise location all around the world, in a fast way using the OpenWeather API.

Use

Add a location

To add a location, click on the orange button at the bottom right of the home page. Then a map will show and you will be able to browse it. When you find the right place, do a long press on the position. A popup will show up and ask you the name of the location. You can close it to choose another place, or validate and you will then go back to the home page, with your new location.

Delete a location

To delete a location, it's very simple. For each location in the list, there is a trash icon. Click on it, and then, valid the popup and that's it.

Get the Forecast

To get the forecast, it's very simple, just click on the chosen location and all the weather informations will show.

Settings

Reset

You can delete all the saved locations just by clicking on the button in the preferences view. A confirmation popup will appear and you will have to validate your choice.

Sow icon

By default, icons are downloaded and show with the forecast. If you want you can turn off the download of these icons.

5 Days Forecast

You can chosse to show the 5 days forecast. It is disabled by default.

Hour Forecast

With the 5 days forecast option, the informations are every 3 hour. To keep a coherence, the application show the forecast for the 5 next days, at the same hour. With this parameter, you can select the hour that you want to display.
